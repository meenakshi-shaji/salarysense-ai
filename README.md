# SalarySense AI

SalarySense AI is an AI-powered career intelligence platform developed using Python, Machine Learning, Streamlit, and SQLite. The application helps users predict industry-level salaries, analyze resumes for ATS optimization, and gain career insights through an interactive dashboard.

The project combines machine learning with a modern web interface to provide a professional and user-friendly experience for students, freshers, and job seekers.

## Features

* AI-based salary prediction system
* Resume ATS analyzer
* Resume skill detection
* Career insights dashboard
* User authentication system
* Prediction history tracking
* PDF salary report generation
* Professional dark-themed UI
* SQLite database integration
* Machine learning model integration

## Technologies Used

* Python
* Streamlit
* Scikit-learn
* Pandas
* SQLite
* Plotly
* ReportLab
* Joblib

## Project Modules

### Salary Predictor

Predicts realistic industry-level salaries based on:

* Job role
* Experience
* Skills
* Certifications
* Company size
* Industry
* Location

### Resume Analyzer

Analyzes resumes and provides:

* ATS score
* Skill detection
* Resume category detection
* Missing skills suggestions
* Resume improvement recommendations

### Career Insights

Provides visual insights related to industries, salaries, and career trends using interactive charts.

## Installation

Clone the repository:

```bash
git clone https://github.com/meenakshi-shaji/salarysense-ai.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
